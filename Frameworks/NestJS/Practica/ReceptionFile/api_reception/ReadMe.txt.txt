Funcionalidad API:
Método POST:
Si cargamos un archivo desde postman http://localhost:3000/file/upload, el mismo se guarda en el disco.
Método GET:
a) Si descargamos un archivo desde postman http://localhost:3000/file/file-1732136789939-985046255.pdf,
el archivo se visualiza y opcionalmente se descarga.
b) Si colocamos en la url de Google: http://localhost:3000/file/file-1732136789939-985046255.pdf el archivo se descarga 
automáticamente.
